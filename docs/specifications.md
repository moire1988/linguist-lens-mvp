# LinguistLens プロジェクト仕様書 & 権限マトリクス

## 1. プロダクト概要
- **名称**: LinguistLens
- **コンセプト**: サイバー・ミニマルなUIで提供する、脱「The 日本人英語」を目指す中上級者向け「生きた英語」抽出・定着SaaS。単なる暗記ゲームを捨て、基本動詞と前置詞のコアイメージ理解によって、最小限の暗記で最大限の表現力を解放する。
- **ターゲット**: YouTube、海外記事、Podcastなどのリアルなコンテンツから、自分のCEFRレベルに合った表現を効率的に学びたい学習者。

### 💡 Core Values（提供する本質的価値）
1. **言語のコア理解（Core Image）**
   - 単語の無機質な丸暗記を避け、`make`, `get`, `take` などの基本動詞や、`off`, `on`, `up` などの副詞・前置詞が持つ「コアイメージ」を掴む手助けをする。
2. **ミニマムな暗記で表現を自由化（Minimum Effort, Maximum Freedom）**
   - ネイティブしか使わないマニアックなイディオムは切り捨て、第二言語として「本当に優先順位の高い、使い回しの効く表現」に限定する。
3. **脱・日本人あるある（Break the "Japanese English"）**
   - `I think...` や `maybe` の多用といった典型的なつまづきから脱却。`difficult` → `tricky`、`sounds good` → `sounds like a plan!` のような、少しこなれた自然な言い回しの習得を促す。
4. **点から面への学習ループ（Comprehensible Input Loop）**
   - 日本語の解説を読むだけでは「使える」ようにはならない。
   - 【収集】リアルなコンテンツから表現を保存 ➔ 【定着】フラッシュカードで短期記憶 ➔ 【統合: Pro限定】覚えた表現を組み込んだ「自分専用のオリジナルAI記事」を生成して多読 ➔ 【習得】文脈の中で何度も出会うことで、自然と口から出る状態（アウトプット）へと昇華させる。

### 🚫 Non-Goals（やらないこと）
- **発音矯正・スピーキング評価**: ELSA Speak等の専用AIツールに譲り、本プロダクトは「表現の引き出しを増やすこと」「ニュアンスの正確な理解」に一点集中する。

## 2. ユーザー権限（ロール）定義
システムは以下の4つの権限レベルでアクセス制御を行う。

1. **Guest (未ログイン)**: お試し利用のユーザー。
2. **Free (無料会員)**: Clerkでアカウント作成済みの一般ユーザー。
3. **Pro (有料会員)**: サブスクリプション課金済みのユーザー（※将来実装予定）。
4. **Admin (管理者)**: ユカCEO。環境変数 `ADMIN_USER_ID` と一致する特権ユーザー。

## 4. データ保存の仕様 (State & DB)
- **Guest**: 基本的に `localStorage` を使用。ブラウザを閉じてもお試しデータが残るようにするが、デバイス間の同期はされない。
- **Free / Pro / Admin**: Clerkの認証情報を利用し、`Supabase` にデータを保存・同期する。PCとスマホ間での学習データのシームレスな同期を実現する。

## 5. UI/UXデザイン原則 (Cyber-Minimal)
- **配色**: ダークモード（または極薄のグリッド背景）を基調とし、アクセントカラーはミニマルに留める。
- **コンポーネント**: 1pxのシャープなボーダー (`border-slate-200` 等) を多用し、角丸やシャドウは上品に。
- **体験**: 「ポップアップで邪魔しない」「引き算のUI」。ユーザーが直感的に「次に何をすべきか」分かるように、プレースホルダーやマイクロコピーで誘導する。
